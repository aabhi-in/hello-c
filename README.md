# Hello World - C Language Project

A simple Hello World program written in C, built with CMake.

## Prerequisites

- GCC or Clang compiler
- CMake 3.10 or higher

## Building

```bash
cd hello-c
mkdir -p build
cd build
cmake ..
make
```

## Running

```bash
./bin/hello_world
```

## Project Structure

```
hello-c/
├── src/
│   └── main.c          # Main program source
├── include/            # Header files (for future use)
├── build/              # Build directory
├── CMakeLists.txt      # CMake configuration
└── README.md           # This file
```
